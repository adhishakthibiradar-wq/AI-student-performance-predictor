
# 🎓 AI Student Performance Predictor

An AI-powered full-stack web application that predicts student performance using Machine Learning and provides personalized study recommendations using Gemini AI.

Features
- Student Performance Prediction
- AI Study Guidance (Gemini AI)
- Random Forest ML Model
- Performance Charts
- Prediction History
- Search & Delete History
- Responsive Modern UI

Tech Stack
- Frontend: React, Vite, Tailwind CSS
- Backend: Node.js, Express.js
- Database: MongoDB Atlas
- ML: Python, Flask, Scikit-learn
- AI: Google Gemini API
